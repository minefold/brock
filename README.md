# Funpack Settings

Generate HTML from definitions. Generate settings from params

## Examples

### Integer

    {
      "name" => "mp_maxrounds",
      "type" => "integer",
      "label" => "Max Rounds",
      "description" => "Maximum number of rounds to play",
      "default" => 5
    }
